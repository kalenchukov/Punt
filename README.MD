# StringRand
Генератор строк случайных символов.

### Использование
```java
// Возвращает строку символов из диапазона 0-1 указанной длинны 2
StringRand.binary(2);
// Результат выполнения: 01

// Возвращает строку символов из диапазона 0-7 указанной длинны 8
StringRand.octal(8);
// Результат выполнения: 31542021

// Возвращает строку символов из диапазона 0-9 указанной длинны 10
StringRand.decimal(10);
// Результат выполнения: 1680320347

// Возвращает строку символов из диапазона 0-9 и A-B указанной длинны 12
StringRand.duodecimal(12);
// Результат выполнения: 41497B790666

// Возвращает строку символов из диапазона 0-9 и A-F указанной длинны 16
StringRand.hex(16);
// Результат выполнения: BFB76A59D4C18A96

// Возвращает строку символов из диапазона a-z указанной длинны 32
StringRand.lower(32);
// Результат выполнения: wpcejmhmosiedsucgostqiknbkjowqxm

// Возвращает строку из строчных букв алфавита указанной длинны 32
StringRand.lower(new RussianAlphabet.LowerCase(), 32);
// Результат выполнения: эвъзнзиылежкымйыыммльенпмюсдгрээ

// Возвращает строку символов из диапазона A-Z указанной длинны 32
StringRand.upper(32);
// Результат выполнения: SDHSKGOLZDKJZEQZZZPVPIJAPYMQYGQU

// Возвращает строку из прописных букв алфавита указанной длинны 32
StringRand.upper(new RussianAlphabet.UpperCase(), 32);
// Результат выполнения: ЛЫУВЖХРУЧТЪФШВНКЕЖЬУЙЫЯЭЮЩРЬХАОЗ

// Возвращает строку символов из диапазонов A-Z и a-z указанной длинны 32
StringRand.alpha(32);
// Результат выполнения: meOnjOUUQwOuyrAnTRiXvmYRAHscrvNm

// Возвращает строку букв алфавита указанной длинны 32
StringRand.alpha(new RussianAlphabet(), 32);
// Результат выполнения: ёъЧПЁПыФоХцИЁзЯЦХФМэШЛБМЫслщДМзР

// Возвращает строку символов из A-Z, a-z и 0-9 указанной длинны 48
StringRand.alnum(48);
// Результат выполнения: oYzqyxYx1Rg1ZwzmCmY4IODbTnClcMnoDHtpkfFxPtcIw9NC

// Возвращает строку из букв алфавита и символов 0-9 указанной длинны 48
StringRand.alnum(new RussianAlphabet(), 48);
// Результат выполнения: 2пСэчъЙдКЛ4кВаЩНЮвщм2ыМХсцВъВОГЭ7л2ЪБ88ЯыбОЬЭ7зы

// Возвращает строку символов из a-z, A-Z, 0-9 и символов !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~ указанной длинны 64
StringRand.graph(64);
// Результат выполнения: kg;1#5KO>zE|V68B2*~hg\zGM*w3:BhI>J$rm;/Qj&,xB3}TVq}/7e}V;a`YO/f_
```
