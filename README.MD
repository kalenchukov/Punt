# Rand
Генератор строк случайных символов.

### Использование
```java
// Возвращает строку символов из диапазона 0-9 указанной длинны 8
Rand.digit(8);
/*
    Результат выполнения: 
    16803203
*/
    
// Возвращает строку символов из диапазона a-z указанной длинны 16
Rand.lower(16);
/*
    Результат выполнения: 
    fbksfuwtallfjjeh
*/

// Возвращает строку символов из диапазона A-Z указанной длинны 24
Rand.upper(24);
/*
    Результат выполнения: 
    RCCVPJJUCJBZHMGPVGNKSDQW
*/

// Возвращает строку символов из диапазонов A-Z и a-z указанной длинны 32
Rand.alpha(32);
/*
    Результат выполнения: 
    meOnjOUUQwOuyrAnTRiXvmYRAHscrvNm
*/

// Возвращает строку символов из A-Z, a-z и 0-9 указанной длинны 40
Rand.alnum(40);
/*
    Результат выполнения: 
    vhlIHe799js74hxKYD6H58oKaNV3m9vpIHPR8sde
*/

// Возвращает строку символов из a-z, A-Z, 0-9 и символов !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~ указанной длинны 48
Rand.graph(48);
/*
    Результат выполнения: 
    TuSYwK8H\O;`bg'6!R3D`^rT%:yEW-6/Dw)6q20XyyM!3SYT
*/
```
